# 昼夜与火源 设计文档

- 日期：2026-07-18
- 状态：核心决策已获用户确认（时长/昼规则/火把形态/提灯取消/篝火可合成/燃烧规则），待实现
- 上游：GDD 核心循环、物品系统与交互重做（已合并）、挥砍三件套

## 1. 主旨

补全 GDD"白昼劳作—黑夜求生"核心循环，同时重做光源体系：**取消随身提灯的免费常亮光圈，火把成为唯一随身光**。黑暗从"氛围"升格为"必须被燃料对抗的敌人"。

## 2. 昼夜时钟

| 阶段 | 真实时长 | 特征 |
|---|---|---|
| 白昼 | 4 分钟 | 环境光明亮（暗幕近乎透明），幻影退场（fade→gone），安宁值自然回升（+1.5/s，光圈规则失效），视野全开 |
| 黄昏 | 1 分钟 | 环境光线性滑向夜晚值，幻影在黄昏末重生于远处，安宁值改用夜间规则 |
| 黑夜 | 3 分钟 | 现行黑暗规则：暗幕全开、光圈决定安宁（火把/篝火/提灯柱光圈内 +5/s，圈外黑暗 -3/s——**无提灯档**） |

- sim 层 `world.clock`（秒，随 tick 累加，模 480）；派生 `phase: 'day' | 'dusk' | 'night'` 与 `phaseK`（阶段内 0..1 进度）
- 渲染层按 phaseK 插值：暗幕 alpha（白 0.06 ↔ 夜 0.94）、环境色调（白昼冷调低饱和绿 ↔ 夜深黑）、迷失特效不受时钟影响
- HUD 加昼夜指示（蒲公英旁一枚小日月盘，转角即时刻）
- 开局时刻：白昼开始后 30 秒（给玩家一个学习白天）

## 3. 光源体系重做

删除：玩家随身常亮光圈（`light.lanternRadiusM` 作为随身光废止）。

| 光源 | 获取 | 光圈 | 寿命 |
|---|---|---|---|
| 手持火把 | 合成：2 木（热键栏选中即点亮） | 2.8m | **不灭**（手持时） |
| 插地火把 | 手持时右键插地 | 2.2m | 90 秒燃尽消失，光圈随余量线性缩至 0.6m 后熄灭 |
| 篝火（新可合成建筑） | 合成：8 木 2 萤石，右键放置 | 5m | 衰减：满燃 120 秒后光圈线性缩小至 1.2m 残烬；靠近按 F（或对篝火右键持木）投 1 木回满。残烬不消失，可再点燃 |
| 提灯柱 | 现行配方不变 | 5m | 永久（远征里程碑目标地位不变） |

- ItemKind 增 `'torch' | 'campfire'`；配方表增两行；放置复用物品系统右键管线
- 夜晚无任何光源时：屏幕近全黑 + 安宁值 -3/s——火把是刚需，出门前备燃料
- 白天手持火把不提供安宁加成（白天规则本就回升），插地火把照常燃烧（劝玩家白天别浪费）

## 4. 出生点

预置篝火删除。出生点放置**古石地标**（不发光、不交互、纯寻路锚点；素材后补，先程序绘制占位）。开局热键栏预置：斧 ×1、火把 ×2（教学善意，保证第一夜活得下去）。

## 5. 幻影昼夜行为

- 白昼开始：幻影 fade→gone（已有状态机复用）
- 黄昏最后 10 秒：于玩家 12m 外重生进入 wander
- 夜晚规则不变（stare 滞回、消散重生）

## 6. 数值（config 增量，全部可调）

clock { dayS: 240, duskS: 60, nightS: 180, startAtS: 30 }
light { torchHeldM: 2.8, torchPlantedM: 2.2, campfireM: 5, campfireEmberM: 1.2, postM: 5（沿用） }
fire { torchBurnS: 90, torchMinM: 0.6, campfireBurnS: 120, campfireFeedWood: 1 }
serenity { dayRegen: 1.5, zoneRegen: 5（沿用）, darkDrain: -3（沿用）, 删 lanternDrain }
recipes += { torch: 2 木 → 1 火把 ×2？（定：2 木 → 2 火把）, campfire: 8 木 + 2 萤石 }

## 7. 呈现

- 火把素材：先程序绘制（木柄+程序火焰粒子沿用篝火火焰做法缩小），千问出图后替换
- 篝火沿用现有已入库素材与程序火焰；残烬态 = 熄焰 + 微红点
- 白昼的手作质感层不变（纸感全天在线）；白天迷雾/颗粒参数可微调但非本里程碑必做

## 8. 范围外

无限地图、区域分布、模仿鸟/守林巨兽（后续里程碑）；火把照明对幻影行为的特殊影响（火把驱赶等）；燃料种类扩展。

## 9. 验证

- 单测：时钟相位派生与回绕、白昼安宁回升/夜晚无光狂掉、火把手持不灭/插地燃尽、篝火衰减与添柴、幻影昼退暮归、配方扣料
- e2e：完整一天流程（白天砍树→黄昏插火把标路→夜晚火把生存→篝火添柴→熬到黎明）
- 上线后浏览器主观验收：黑夜无火把的恐慌感是否成立
