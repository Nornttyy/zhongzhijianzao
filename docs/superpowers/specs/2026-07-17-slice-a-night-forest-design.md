# 《森之低语》切片 A ——"夜之切片"设计文档

- 日期：2026-07-17
- 状态：设计已获用户确认，待实现与试玩验收
- 上游文档：`docs/GDD-whispers-of-the-woods.md`（全案基准）

## 1. 背景与目的

《森之低语》是 2D 俯视角多人合作"微恐治愈"游戏，发布目标为 itch.io 网页版。全案横跨十余个子系统，按里程碑拆解落地。本切片是第一个里程碑：**用最小内容验证本作最大的未知数——"迷失的微恐 + 点灯的治愈"这套氛围与手感是否成立**。约 5 分钟的完整微型体验。

技术路线已确认：TypeScript + PixiJS + Vite 的网页版（itch.io 浏览器可直接玩、包体小、本工作区内可自测）；若未来上 Steam，再将验证过的玩法迁移至 Godot。

## 2. 范围

### 做

1. 玩家移动（WASD/方向键），随身提灯光圈；圈外为黑暗与迷雾
2. 安宁值系统（0–100）：按所处光照档位每秒结算；低于 30% 触发"迷失状态"（屏幕边缘起雾、画面降饱和、音频闷化），回升至 40% 解除（滞回防抖）
3. 采集：砍低语木（金色萤火虫粒子）、挖萤石矿；采集有 1.2 秒动作时间与进度反馈；右下角计数背包 UI（仅两种资源，无格子系统）
4. 迷途幻影 ×1：远处游荡→停下注视（压迫音渐强）→玩家靠近至 6m 内淡出消散+轻叹，随机换位重生；不攻击、不追击
5. 篝火营地；合成目标：10 低语木 + 5 萤石 → 提灯柱 ×1，放置后成为永久光源、扩大安全区；完成后弹出柔和的达成提示
6. 程序生成环境音（WebAudio）：粉噪声风声、低频低鸣（幻影靠近时增益）、砍木敲击声、轻叹；迷失状态全局低通滤波

### 不做

多人联机、完整合成/熔炉系统、四大区域、模仿鸟/守林巨兽/绒尾兔、语音录制与变声、存档、剧情引导脚本、手柄支持。

## 3. 呈现方式

《饥荒》式混合视角：地面为俯视平面，角色与物件为正面"立牌"立绘，水平翻转表现朝向；渲染按 y 坐标排序制造前后遮挡。每个素材只需一张去底立绘，利于 AI 出图风格一致。

逻辑坐标单位为"米"，渲染换算 48px/m（可调）。

## 4. 系统设计

### 4.1 光照与黑夜

全屏暗幕，提灯（随身）、篝火、提灯柱在暗幕上开半径不等的柔边光洞，光洞边缘做噪声抖动模拟火光呼吸。光圈外叠加滚动噪声迷雾层。

### 4.2 安宁值

每逻辑帧按玩家所处最高光照档位结算（数值见 §6）。迷失状态为纯表现层 Debuff（本切片不影响操作）。

### 4.3 采集与背包

低语木/萤石矿为可交互实体，进入交互半径显示提示，按 E 采集。每棵低语木可砍 4 次（每次 1 木），砍完变树桩（不可再采）；每处萤石矿可挖 5 次（每次 1 萤石）。场景布置 6 棵树、3 处矿（资源冗余，避免软锁）。

### 4.4 合成与放置

靠近篝火且资源足够时出现合成提示，按 E 消耗资源获得提灯柱并进入放置模式（预览跟随角色前方 1.5m，随朝向），再按 E 确认落地。统一交互键：E（采集/合成/放置）。

### 4.5 输入

移动 WASD/方向键；**采集 = 鼠标左键**（长按连砍；移动不打断、砍击中减速；挥砍朝向鼠标所在侧）⁴；合成/放置 = E。³

## 5. 技术架构

- Vite + TypeScript + PixiJS v8
- 目录：`src/sim/`（纯逻辑，固定步长 30Hz，零 Pixi 依赖）、`src/render/`（Pixi 场景、插值渲染、特效）、`src/audio/`（WebAudio 合成）、`src/config.ts`（全部数值集中于此）、`src/assets/`（素材加载，缺失时自动回退程序绘制占位图形）
- **sim/render 分离即联机预留**：未来 sim 层可整体迁移到服务器，render 只消费状态快照
- 素材管线：AI 原图入 `assets/raw/`，脚本去底（白底泛洪抠图；黑底素材直接用加法/滤色混合）、裁边、命名后入 `public/assets/`；游戏内叠全局色调层统一色感

## 6. 数值初稿（config.ts，全部可调）

| 项 | 值 |
|---|---|
| 安宁值上限 / 迷失阈值 / 解除阈值 | 100 / 30 / 40 |
| 提灯光圈内（野外） | -0.5/秒 |
| 光圈外黑暗中 | -3/秒 |
| 幻影注视范围内额外 | -2/秒 |
| 篝火/提灯柱光圈内 | +5/秒 |
| 光圈半径：提灯 / 篝火 / 提灯柱 | 3.5m / 6m / 5m |
| 幻影注视范围 / 消散距离 | 8m / 6m |
| 移动速度 | 4 m/s |
| 采集动作时长 | 1.2 秒 |
| 提灯柱配方 | 10 低语木 + 5 萤石 |

## 7. 验证

- vitest 单测：安宁值各档位结算与滞回、采集判定与资源耗尽、合成资源校验与放置状态机
- 构建冒烟：`vite build` 通过；如环境可安装无头浏览器则加截图自查
- **验收标准**：新玩家 5 分钟内完成"感到黑暗压力→采集→遭遇幻影→合成并放置提灯柱"，全程无报错、无软锁；用户在浏览器试玩后主观认可"微恐+治愈"氛围成立

## 8. 术语与文档修订

全案统一使用"安宁值"（GDD 原文两处"理智"已在转录版标注修正）；"血肉暗红"认定为《骨木》旧稿残留，夜晚点缀色由美术阶段确定。
